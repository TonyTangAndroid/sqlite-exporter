README
------------------------------

In this example, the export configuration has been set up so that:
- For all exports,
  - use 'name' as the primary field for the 'Animals' table export
  - exclude the 'Secrets' table
  - exclude the 'id' field in the 'Employees' table
- For the XML export,
  - exclude the 'quantity' field in the 'Animals' table
- For the JSON export,
  - exclude the 'salary' field in the 'Employees' table
